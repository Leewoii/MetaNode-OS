# MetaNode OS Private Core Deployment

## Architecture

- Public launcher repo: `apps/launcher`.
- Cloudflare Worker access gate: `apps/gate-worker`.
- Koyeb/Render distribution proxy: `apps/distribution-api`.
- Private core repo/release: Docker image bundle generated by `scripts/build-core-bundle.mjs`.

The launcher never receives a GitHub token. It gets a short-lived runtime grant from the Worker, asks the Worker for a short-lived download token, downloads release assets through the distribution API, verifies the manifest hashes, loads Docker images, and starts the core compose stack.

## Required Secrets

Cloudflare Worker:

- `ADMIN_TOKEN`
- `APP_GRANT_SECRET`
- `DISTRIBUTION_SHARED_SECRET`

Distribution API:

- `GITHUB_TOKEN`
- `GITHUB_OWNER`
- `GITHUB_REPO`
- `DISTRIBUTION_SHARED_SECRET`

Private core release CI:

- `METANODE_RELEASE_PRIVATE_KEY_PEM`

Launcher build/runtime:

- `METANODE_GATE_URL`
- `METANODE_DISTRIBUTION_URL`
- `METANODE_RELEASE_PUBLIC_KEY_BASE64`
- optional `METANODE_USER_ID`

## Flow

1. Launcher creates a local device ID.
2. Launcher calls `POST /v1/access/check`.
3. Worker derives the caller IP from Cloudflare request metadata.
4. If not approved, Worker stores a pending request and launcher shows the pending message.
5. Admin approves `user + IP + device`.
6. Launcher receives a short-lived grant.
7. Launcher calls Worker `POST /v1/core/download-token`.
8. Launcher downloads private release assets through the distribution API.
9. Launcher verifies `core-manifest.sig` and `core-manifest.json` hashes.
10. Launcher runs `docker load`, `docker compose up -d`, and optional Docker Ollama fallback.
11. Core API startup verifies `METANODE_RUNTIME_GRANT` through the Worker before listening.

## Notes

- This is tamper resistance, not unbreakable DRM.
- If the full core runs on the user machine, a determined user can inspect or copy runtime artifacts.
- The important protection is that copied core images still need a valid short-lived runtime grant to start.
- Replace `apps/launcher/src-tauri/icons/icon.ico` with the real MetaNode OS icon before public release.
